# EduSmart

EduSmart is a web-based educational platform developed as an academic project.

## Main Features

- Teacher and student registration and login
- AI-based quiz generation
- Automatic quiz assignment
- Quiz solving and grading
- Reports and results
- Admin dashboard
- School / University mode
- LTI 1.1 integration with Moodle / MAMA
- OpenAI API integration
- YouTube-based learning support

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- OpenAI API
- LTI 1.1
- OAuth 1.0
- cPanel Hosting
- Apache Server

## System Overview

The system allows teachers to create quizzes using AI, assign them to students, and view reports and grades.  
Students can log in, solve assigned quizzes, and receive results.  
The system also supports integration with Moodle / MAMA using LTI 1.1, allowing users to access EduSmart directly from an LMS environment.

## External Integrations

- OpenAI API – automatic quiz question generation
- Moodle / MAMA – LMS integration using LTI 1.1
- YouTube – related learning videos for feedback and support
